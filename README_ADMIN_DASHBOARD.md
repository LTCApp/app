# لوحة تحكم الإدارة - سوق الجملة

## نظرة عامة
تم إنشاء لوحة تحكم شاملة لإدارة تطبيق سوق الجملة. تتضمن اللوحة جميع الميزات الأساسية لإدارة المتجر الإلكتروني بشكل فعال ومتكامل.

## الميزات الرئيسية

### 🔐 نظام تسجيل الدخول الآمن
- **اسم المستخدم:** `admin`
- **كلمة المرور:** `admin123`
- واجهة دخول أنيقة مع حماية كاملة
- إمكانية إظهار/إخفاء كلمة المرور

### 📊 لوحة الإحصائيات الرئيسية
- **بطاقات الإحصائيات:** عرض مباشر للمبيعات، الطلبات، العملاء، والمنتجات
- **الرسوم البيانية التفاعلية:**
  - رسم بياني خطي لتطور المبيعات الشهرية
  - رسم بياني دائري لتوزيع المنتجات حسب الفئات
- **الطلبات الحديثة:** عرض آخر الطلبات مع إمكانية المتابعة السريعة

### 🛍️ إدارة المنتجات
- **عرض شامل:** جميع المنتجات في شبكة منظمة
- **إضافة منتج جديد:** نموذج متكامل مع:
  - اسم المنتج والوصف
  - الفئة والسعر
  - الكمية المتوفرة
  - رفع صورة المنتج
- **تعديل المنتجات:** إمكانية تعديل جميع تفاصيل المنتج
- **حذف المنتجات:** مع تأكيد الحذف لمنع الأخطاء
- **البحث والتصفية:**
  - البحث بالاسم أو الوصف
  - تصفية حسب الفئة
  - تصفية حسب حالة المخزون
- **حالات المخزون الذكية:**
  - متوفر (أخضر): أكثر من 20 قطعة
  - مخزون منخفض (أصفر): أقل من 20 قطعة
  - نفد المخزون (أحمر): 0 قطعة

### 📋 إدارة الطلبات
- **عرض تفصيلي:** جميع الطلبات مع معلومات شاملة
- **حالات الطلبات:** 6 حالات مختلفة قابلة للتحديث
  - قيد المراجعة
  - مؤكد
  - قيد التحضير
  - خارج للتوصيل
  - تم التسليم
  - ملغي
- **تفاصيل الطلب:** نافذة منبثقة تعرض:
  - تفاصيل العميل الكاملة
  - قائمة المنتجات والكميات
  - إجمالي المبلغ
  - عنوان التوصيل
- **البحث والتصفية:**
  - البحث برقم الطلب أو اسم العميل
  - تصفية حسب حالة الطلب
  - تصفية حسب التاريخ
- **إحصائيات سريعة:** عدد الطلبات حسب الحالة

### 👥 إدارة المستخدمين
- **ملفات تعريف شاملة:** معلومات مفصلة لكل مستخدم
- **إحصائيات المستخدم:**
  - إجمالي الطلبات
  - إجمالي المشتريات
  - تاريخ الانضمام وآخر دخول
- **أنواع المستخدمين:**
  - عادي: عملاء التجزئة
  - مميز: عملاء VIP
  - جملة: عملاء الشراء بالجملة
- **إدارة الحالات:**
  - نشط
  - غير نشط
  - محظور
- **إجراءات الإدارة:**
  - حظر/إلغاء حظر المستخدمين
  - عرض التفاصيل الكاملة
- **البحث والتصفية:**
  - البحث بالاسم، البريد، أو رقم الهاتف
  - تصفية حسب حالة المستخدم
  - تصفية حسب نوع المستخدم

## التصميم والواجهة

### 🎨 نمط التصميم
- **الألوان:** نفس نمط التطبيق الأساسي (الأحمر التدرجي)
- **الخط:** Cairo لدعم اللغة العربية بشكل مثالي
- **التخطيط:** استجابة كاملة لجميع أحجام الشاشات
- **الرسوم المتحركة:** تأثيرات سلسة ومتقنة

### 📱 التصميم المتجاوب
- **الهواتف الذكية:** تخطيط محسن للشاشات الصغيرة
- **الأجهزة اللوحية:** استغلال أمثل للمساحة المتوسطة
- **أجهزة سطح المكتب:** عرض شبكي متقدم للبيانات

### ⚡ تجربة المستخدم
- **التنقل السريع:** أزرار تبويب سهلة الوصول
- **البحث الفوري:** نتائج فورية أثناء الكتابة
- **التأكيدات الذكية:** رسائل تأكيد للعمليات المهمة
- **التحديثات المباشرة:** تحديث البيانات فورياً

## الملفات المرفقة

### 📄 الملفات الأساسية
- `admin.html` - الصفحة الرئيسية للوحة التحكم
- `admin-style.css` - أنماط CSS للتصميم والتخطيط
- `admin-script.js` - وظائف JavaScript والتفاعل

### 🔧 المتطلبات التقنية
- متصفح حديث يدعم ES6+
- اتصال بالإنترنت لتحميل المكتبات الخارجية:

  - Font Awesome للأيقونات
  - Chart.js للرسوم البيانية

## كيفية الاستخدام

### 🚀 البدء السريع
1. افتح ملف `admin.html` في المتصفح
2. سيظهر نموذج تسجيل الدخول
3. أدخل بيانات الدخول:
   - **المستخدم:** `admin`
   - **كلمة المرور:** `admin123`
4. اضغط "تسجيل الدخول"

### 📋 التنقل في اللوحة
- **الإحصائيات:** العرض الافتراضي مع ملخص شامل

- **الطلبات:** متابعة ومعالجة الطلبات
- **المستخدمين:** إدارة قاعدة العملاء

### ⚙️ الوظائف المتقدمة
