## 🚀 كيفية الاستخدام

### ⚡ التشغيل السريع
```bash
# افتح الواجهة الجديدة مباشرة
enhanced-index.html
```

### 🔄 التنقل بين الصفحات
- من الواجهة الجديدة → السلة: انقر على أيقونة السلة العائمة
- من الواجهة الجديدة → الحساب: انقر على "حسابي" في الشريط السفلي  
- من الواجهة الجديدة → الإدارة: للمشرفين فقط

## 🎮 دليل التفاعل السريع

### ⌨️ اختصارات مفيدة
```
Ctrl + K  → البحث السريع
مسافة    → تشغيل/إيقاف الموسيقى
T        → تغيير السمة
E        → تبديل المؤثرات
```

### 🖱️ تفاعل الماوس
- **حرك الماوس** → تتبع الأضواء
- **اهوفر على البطاقات** → تأثيرات 3D
- **انقر الأزرار** → انفجارات بصرية
- **اسحب العناصر** → تأثيرات متقدمة
