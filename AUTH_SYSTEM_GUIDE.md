# 🚀 نظام تسجيل الدخول وإنشاء الحساب المتكامل

## 📋 نظرة عامة
تم إنشاء نظام مصادقة متطور وشامل يدعم تسجيل الدخول وإنشاء الحساب بطريقتين:
- **رقم الموبايل** مع التحقق عبر كود OTP
- **البريد الإلكتروني** مع كلمة المرور

## 📁 الملفات المُنشأة

### صفحات التسجيل:
- `login.html` - صفحة تسجيل الدخول الاحترافية
- `register.html` - صفحة إنشاء حساب جديد
- `auth-style.css` - ملف التصميم الشامل للصفحتين

### ملفات JavaScript:
- `auth-system.js` - النظام الأساسي للمصادقة
- `login-handler.js` - معالج تسجيل الدخول
- `register-handler.js` - معالج التسجيل
- `main-auth.js` - إدارة المصادقة في الصفحة الرئيسية

### التحديثات:
- `index.html` - تم ربط النظام بالصفحة الرئيسية
- `style.css` - إضافة تصميمات أزرار تسجيل الدخول والقوائم

## ✨ الميزات الرئيسية

### 🔐 تسجيل الدخول
- **بالموبايل**: كشف تلقائي للمستخدمين الحاليين
- **بالإيميل**: تسجيل دخول فوري بكلمة المرور
- **تحقق OTP**: نظام كود التحقق المؤقت (6 أرقام)
- **تذكرني**: حفظ بيانات تسجيل الدخول
- **استرداد كلمة المرور**: رابط لاستعادة كلمة المرور

### 📝 إنشاء الحساب
- **التسجيل بالموبايل**: مع تحقق OTP إجباري
- **التسجيل بالإيميل**: إنشاء فوري بدون تحقق
- **التحقق من قوة كلمة المرور**: مؤشر بصري للقوة
- **منع التكرار**: فحص الإيميل/الموبايل المكرر
- **الشروط والأحكام**: موافقة إجبارية

### 🎨 التصميم
- **تصميم عصري**: ألوان متدرجة وتأثيرات بصرية
- **تجاوب كامل**: يعمل على جميع الأجهزة
- **رسوم متحركة**: انتقالات سلسة ومؤثرات جذابة
- **خلفية متحركة**: دوائر عائمة في الخلفية
- **أيقونات FontAwesome**: أيقونات احترافية

### 📱 تجربة المستخدم
- **إدخال OTP ذكي**: انتقال تلقائي بين الخانات
- **لصق الكود**: دعم لصق كود التحقق من الحافظة
- **مؤقت العد التنازلي**: 60 ثانية مع إمكانية الإعادة
- **رسائل التوجيه**: إشعارات واضحة للمستخدم
- **حفظ الجلسة**: البقاء مسجلاً عبر الجلسات

## 🔧 كيفية الاستخدام

### للمستخدمين الجدد:
1. **افتح `index.html`** في المتصفح
2. **اضغط "حساب جديد"** في أعلى الصفحة
3. **اختر طريقة التسجيل**: موبايل أو إيميل
4. **أكمل البيانات** وأرسل النموذج
5. **أدخل كود التحقق** (للتسجيل بالموبايل)
6. **مبروك!** تم إنشاء حسابك

### للمستخدمين الحاليين:
1. **افتح `index.html`** في المتصفح
2. **اضغط "دخول"** في أعلى الصفحة
3. **اختر طريقة الدخول**: موبايل أو إيميل
4. **أدخل بياناتك**
5. **أدخل كود التحقق** (عند الحاجة)
6. **أهلاً بعودتك!** تم تسجيل الدخول

## 🧪 اختبار النظام

### كيفية رؤية كود OTP:
- **في وحدة التحكم**: `F12` → Console → ابحث عن الكود
- **في الإشعارات**: سيظهر إشعار بالكود لمدة 6 ثوانٍ
- **الكود الافتراضي**: 6 أرقام عشوائية

### حسابات تجريبية:
يمكنك إنشاء حسابات تجريبية باستخدام:
- **موبايل**: +20 1234567890
- **إيميل**: test@example.com
- **كلمة المرور**: 123456789

- `rememberedEmail` - الإيميل المحفوظ

## 🎯 المميزات المتقدمة

### 🔄 إدارة الحالة:
- **تحديث تلقائي**: عند تغيير حالة تسجيل الدخول
- **مزامنة بين التبويبات**: عبر localStorage events
- **فحص دوري**: كل 30 ثانية للتأكد من الحالة

## 📞 الدعم والمساعدة

### رسائل الخطأ الشائعة:
- "رقم الموبايل مسجل بالفعل" → استخدم تسجيل الدخول
- "كود التحقق غير صحيح" → تحقق من الكود في وحدة التحكم
- "كلمات المرور غير متطابقة" → تأكد من التطابق
- "يجب الموافقة على الشروط" → فعّل خانة الموافقة

### نصائح للاستخدام الأمثل:
1. **استخدم أرقام موبايل حقيقية** للاختبار
2. **اختر كلمات مرور قوية** (8 أحرف + أرقام + رموز)
3. **فعّل "تذكرني"** لسهولة الدخول اللاحق
4. **تحقق من وحدة التحكم** لرؤية أكواد OTP
