# 🛒 تطبيق سوق الجملة - نظام الحسابات المتكامل

## 📱 وصف المشروع المحدث

تطبيق جوال احترافي لسوبر ماركت "سوق الجملة" مع نظام حسابات متكامل يدعم التسجيل عبر رقم الموبايل أو حساب جوجل. التطبيق مصمم خصيصاً للهواتف المحمولة باللون الأحمر الغالب مع تجربة مستخدم استثنائية.

## 🆕 الميزات الجديدة - صفحة الحساب

## 🚀 التشغيل والاستخدام

### متطلبات النظام
- متصفح حديث (Chrome, Firefox, Safari, Edge)
- دعم JavaScript ES6+
- اتصال بالإنترنت (للخطوط والأيقونات وGoogle OAuth)
- دعم localStorage

### خطوات التشغيل
1. افتح ملف `index.html` في أي متصفح
2. انقر على "حسابي" في التنقل السفلي
3. اختر بين تسجيل الدخول أو إنشاء حساب جديد
4. اتبع التعليمات لإكمال التسجيل
5. استمتع بتجربة التطبيق الكاملة!

### كيفية اختبار OTP
```javascript
// سيتم طباعة كود OTP في console للاختبار

// أو يمكنك استخدام الكود الافتراضي: 123456
```

## 📱 دليل الاستخدام

### 🔑 التسجيل الجديد
1. اضغط على "إنشاء حساب جديد"
2. أدخل الاسم الكامل
3. اختر كود الدولة ورقم الموبايل
4. أدخل البريد الإلكتروني (اختياري)
5. أنشئ كلمة مرور قوية
6. وافق على الشروط والأحكام
7. أدخل كود OTP المرسل لهاتفك
8. استمتع بحسابك الجديد!

### 🚪 تسجيل الدخول
1. اختر "تسجيل الدخول"
2. أدخل رقم الموبايل
3. أدخل كود OTP المرسل
4. مرحباً بعودتك!

### 🌐 تسجيل الدخول مع Google
1. اضغط على "متابعة مع جوجل"
2. اختر حسابك في Google
3. تم! حساب جاهز فوراً
