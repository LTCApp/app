# تعليمات سريعة - ميزة الإعدادات

## كيفية الوصول للإعدادات

### 1. طريقة التنقل المباشر
- افتح لوحة التحكم
- انقر على تبويب **"الإعدادات"** في شريط التنقل

### 2. من القائمة الشخصية
- انقر على صورة الملف الشخصي في الأعلى
- اختر **"الإعدادات"** من القائمة المنسدلة

---

## الميزات الأساسية

### 🏪 إدارة حالة المتجر

#### فتح المتجر
```
✅ انقر على زر "فتح الفرع"
✅ سيظهر مؤشر أخضر + نص "مفتوح"
```

#### إغلاق مؤقت
```
🕐 انقر على "إغلاق مؤقت"
🕐 اختر من الخيارات:
   • ساعة واحدة (تلقائي)
   • وقت محدد (اختر التاريخ والوقت)
   • باقي اليوم (يفتح غداً)
🕐 أضف سبب الإغلاق (اختياري)
🕐 انقر "تأكيد الإغلاق"
```

#### إغلاق كامل
```
🔒 انقر على زر "إغلاق الفرع"
🔒 سيظهر مؤشر أحمر + نص "مغلق"
```

### 🕒 إدارة مواعيد العمل

```
1️⃣ لكل يوم من الأسبوع:
   • استخدم المفتاح لتفعيل/إلغاء اليوم
   • حدد وقت البداية (مثال: 08:00)
   • حدد وقت النهاية (مثال: 22:00)

2️⃣ انقر "حفظ ساعات العمل"

💡 نصيحة: الجمعة غير مفعل افتراضياً
```

### 🔔 إعدادات التنبيهات

```
🔊 الأصوات:
   • صوت الطلبات الجديدة ✓/✗
   • تنبيه نفاد المخزون ✓/✗

📱 التنبيهات المرئية:
   • تنبيهات سطح المكتب ✓/✗
   • عدادات التنبيه ✓/✗
```

### ⚙️ إعدادات النظام

#### العملة والأسعار
```
💰 رمز العملة: ر.س / $ / € / £
💰 الحد الأدنى للطلب: (مثال: 50)
💰 رسوم التوصيل: (مثال: 15)
```

#### إعدادات التوصيل
```
🚚 وقت التحضير: (بالدقائق)
🚚 وقت التوصيل: (بالدقائق)  
🚚 نطاق التوصيل: (بالكيلومتر)
```

## حفظ الإعدادات

### حفظ تلقائي
- مواعيد العمل: زر "حفظ ساعات العمل"
- حالة المتجر: يحفظ تلقائياً عند التغيير

### حفظ شامل
```
💾 انقر "حفظ جميع الإعدادات" لحفظ:
   ✓ إعدادات التنبيهات
   ✓ إعدادات النظام
   ✓ مواعيد العمل
```

### استعادة افتراضية
```
🔄 انقر "استعادة الافتراضية"
⚠️  تحذير: سيتم فقدان جميع التخصيصات
```

## رسائل الحالة

### مؤشرات المتجر
- 🟢 **أخضر**: مفتوح
- 🟡 **أصفر**: مغلق مؤقتاً (مع عداد الوقت)
- 🔴 **أحمر**: مغلق

### رسائل النجاح
- ✅ "تم فتح الفرع بنجاح"
- ✅ "تم إغلاق الفرع مؤقتاً"
- ✅ "تم حفظ ساعات العمل بنجاح"
- ✅ "تم حفظ جميع الإعدادات بنجاح"

## نصائح مهمة

### 🎯 لأفضل النتائج

1. **اختبر الإعدادات**: استخدم ملف `settings_test.html` للتأكد من عمل كل شيء

2. **النسخ الاحتياطي**: احفظ إعداداتك المخصصة قبل الاستعادة

3. **أوقات العمل**: تأكد من ضبط أوقات منطقية (البداية أقل من النهاية)

4. **الإغلاق المؤقت**: استخدمه للصيانة أو استراحة الغداء

5. **رسوم التوصيل**: احسبها بناء على تكلفة الوقود والمسافة

### 📱 للأجهزة المحمولة

- جميع الإعدادات متجاوبة مع الشاشات الصغيرة
- استخدم الاتجاه العمودي للحصول على أفضل تجربة
- المفاتيح والأزرار محسنة للمس

### 🔒 الأمان

- فقط المدراء المعتمدون يمكنهم الوصول للإعدادات
- جميع التغييرات محفوظة تلقائياً
- لا تشارك بيانات الدخول الإدارية

## حل المشاكل السريع

### ❓ المشكلة: لا يمكن الوصول للإعدادات
**✅ الحل**: تأكد من تسجيل الدخول كمدير

### ❓ المشكلة: لا تحفظ الإعدادات
**✅ الحل**: انقر على أزرار الحفظ المناسبة

### ❓ المشكلة: الإغلاق المؤقت لا يعمل
**✅ الحل**: تأكد من ضبط وقت مستقبلي

### ❓ المشكلة: مواعيد العمل غير صحيحة
**✅ الحل**: تأكد من تفعيل الأيام وضبط الأوقات بشكل صحيح
